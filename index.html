What is this?

Brainwave Runner is an interactive visualization where a simple game level is generated from polysomnography (PSG) recordings. Each second of EEG becomes one column of terrain in the obstacle course.

How it works (high level)

  1. Take an EEG channel like EEG Fpz-Cz from a Sleep-EDF PSG file.
  2. Split it into short windows and compute band power: delta, theta, alpha, beta.
  3. Normalize those values and map them into ground height, gaps, and obstacles.
  4. Draw that as a scrolling 2D level in the canvas above.
